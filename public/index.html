Instructions

Complete the following three steps by entering relevant information.
Inputs left blank are assumed to be unused and will not be used in calculations.

About

This calculator simulates 1000 iterations of combat in OSRS.
The resulting graph displays the number of game ticks needed to end combat.
Combat formulas used are either exact or the best known approximations.
